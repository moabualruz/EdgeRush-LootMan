# Web Dashboard Spec

## Overview

Web-based dashboard providing raider and administrative interfaces for FLPS transparency and guild management.

## Current Problem

No user-facing interface for FLPS data. Raiders cannot self-service check scores, history, or performance. Officers must manually communicate all information.

## Solution

- React + TypeScript frontend with Material-UI
- OAuth2 authentication (Discord + Battle.net)
- Real-time updates via WebSocket
- Responsive design for mobile access
- Admin panel for guild configuration

## Key Features

- Personal FLPS dashboard with detailed breakdowns
- Guild leaderboard with filtering
- Loot history with RDF tracking
- Wishlist with upgrade values
- Performance metrics from Warcraft Logs
- Admin configuration panel
- Behavioral action and loot ban management
- Real-time score updates
